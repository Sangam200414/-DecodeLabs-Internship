# Responsive Web Interface

🚀 This project was created as part of my Full Stack Development Internship at DecodeLabs.

## 📌 Project Overview
A fully responsive web interface built using HTML, CSS, and JavaScript.  
The goal of this project is to demonstrate responsive design principles, semantic HTML structure, and basic JavaScript interactivity.

## ✨ Features
- Semantic HTML5 structure
- Mobile-first responsive design
- CSS Grid & Flexbox layout
- Interactive popup using JavaScript
- Clean and modern UI

## 🛠 Tech Stack
- HTML5
- CSS3
- JavaScript
